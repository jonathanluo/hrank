Oracle Coherence REST Examples

Coherence REST provides easy access to Coherence caches and cache entries over the HTTP protocol.

The examples below utilize JavaScript to access, create and modify Coherence cache entries, store and retrieve binary and JSON data via "pass-through" as well as responding to Server Sent Events (SSE).
